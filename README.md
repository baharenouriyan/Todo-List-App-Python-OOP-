# Todo List App (Python OOP)

A simple command-line Todo List application built with Python using Object-Oriented Programming.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using files

## Concepts Used
- Classes & Objects
- File Handling
- Exception Handling
- Loops
- User Input

## How to Run
`bash
python todo.py
